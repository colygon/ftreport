Data Pipeline

Brevo Email List Segmentation

Built a complete pipeline to segment 189,056 leads into targetable audiences, filter suppressions, and export clean lists for Brevo import. Lists delivered to Aqueel Ali at Funding The Commons.

Pipeline Steps

Step 1

Built 46 Interest Segments

Used keyword matching on event titles to tag leads into segments like AI (12,008), Neurotech (2,243), Music (9,409), Ethereum (7,680), and 42 more categories.

Step 2

Pulled SendGrid Suppressions

Fetched all bounces, invalid emails, spam reports, global unsubscribes, and group unsubscribes from the SendGrid API — 8,606 total emails to block out of 189,056.

Step 3

Created Tagged Brevo List

180,457 clean contacts with EMAIL, FIRSTNAME, TAGS columns — semicolon-separated segments (e.g. AI;Crypto;Founders;Hackathons) preserving all overlapping tags.

Step 4

Created Non-Overlapping Lists

Each email assigned to exactly one segment for simplified sending workflows at FTC.

Step 5

Identified Untagged Contacts

50,594 leads with no segment membership — nearly all only attended general-topic events (ice tank challenge, office hours, etc.).

Core Features

Event Selection

Dropdown of all events from the database with search. Shows event title, date, location, and cover image pulled from Luma.

Two Email Designs

Classic — Luma-style personal invitation with event card, date/time, location, and RSVP button.
FTC Branded — Custom template with hero image, red CTA buttons, speaker headshots grid, and two blurb variants.

Audience Targeting

Send to any of 46 segments, past attendees of similar events, with cross-segment deduplication — no one gets emailed twice for the same event.

Deliverability & Tracking

Deliverability Optimizations

Click/open tracking disabled to reduce spam signals. Custom plain text version. HMAC-signed unsubscribe links via SendGrid ASM groups. Speaker images hosted on Vercel.

Send Tracking

Every invite recorded in sent_invites table. Full send history log with segment, count, and timestamp. ML-scored similar event recommendations.

Send Modes

Test — Single email for preview. Bulk — Entire segment with confirmation modal showing recipient count. Custom subject lines per design.
